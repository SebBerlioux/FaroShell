# Projet FaroShell

## Objectifs

Ce projet a pour but de créer un mini shell en langage C dans lequel les commandes suivantes seront possibles à utilisées :
- mkdir  ✓  (sans options)
- cd     ✓  (sans options)
- ls     ✓  (avec opt)
- cat    ✓  (sans options)
- mv     ✓  (sans options)
- cp     -> Arthur
- rm     ✓  (Options Disponible : -r -f )
- du     ✓  (sans options)
- chmod  ✓  
- chgrp  ✓ (il faut un vrai linux pour la test)
- echo   ✓  (sans options)
- su     ->  Guillaume
