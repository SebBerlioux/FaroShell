# Projet FaroShell

## Objectifs

Ce projet a pour but de créer un mini shell en langage C dans lequel les commandes suivantes seront possibles à utilisées :
- mkdir
- cd
- ls
- cat
- mv
- cp
- rm
- du
- chmod
- chgrp
- echo
- su
