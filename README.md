# Projet FaroShell

## Objectifs

Ce projet a pour but de créer un mini shell en langage C dans lequel les commandes suivantes seront possibles à utilisées :

- cd     ✓
- mkdir  ✓  (sans options)
- ls     ✓  (avec options)
- cat    ✓  (sans options)
- mv     ✓  (Options disponibles : -i -v)
- cp     ✓  (sans options)
- rm     ✓  (Options disponibles : -r -f )
- du     ✓  (sans options)
- chmod  ✓  
- chgrp  ✓
- echo   ✓  (sans options)
